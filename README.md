# Read Me

1. start containers in background (only use images)

   `docker-compose up -d`

1. config db

   `docker-compose exec memory_app_back rake db:create rake db:migrate`

1. tests

   `docker-compose exec memory_app_back rspec`

1. access

   `http://localhost:3000`
